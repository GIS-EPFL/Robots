# Contributors Guide

Git via VSCode terminal:

### Git Pull
* git add --all
* git commit -m "PREFIX message"
* git push

### Branches -> Create and Move to a new Branch
* git branch <name>
* git checkout <name>

### Commit PREFIX rules
* ADD "" - new feature
* FIX "" - bug fix
* DOC "" - documentation
* TEST "" - tests
* TUT "" - tutorials
* MISC "" - unknown
